What is Forge?

  • Single-file HTML outputs — diagrams, galleries, guides, slides
  • Offline-first — works from file://, no server needed
  • 6 specialized skills — one CLI command per artifact type
  • 6 brand aesthetics — match your project's identity
  • Zero runtime deps — just CDN fonts and optional Mermaid
  • Claude-native — prompt → polished artifact in seconds

Guides & Slides

  • forge-guide — multi-tab split-file docs with navigation
  • forge-slides — scroll-snap pitch decks (like this one)
  • forge-epic — issue-linked visual analysis pages
  • forge-md — render existing markdown to themed HTML

All outputs are single-file, self-contained, and work offline.
