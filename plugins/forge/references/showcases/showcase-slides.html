under the hood

Technical Details

Zero runtime deps file:// safe CDN fonts only
Single HTML output Inline CSS Optional Mermaid

All CSS and JS is inlined into the output file. No build step, no bundler, no npm install. Just fonts from Google Fonts CDN (self-hostable) and optional Mermaid for diagrams.
