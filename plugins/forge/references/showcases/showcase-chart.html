Distribution Rules
Inline (generation source)
base/, aesthetics/, slide-templates/ — always inlined into output <style>. No runtime deps.
Link (multi-diagram)
gallery-templates/, fgraph templates — link _shared/ for 2+ diagrams in same doc.
Per-Skill References
skills/<name>/references/*.md — read on-demand via skill directive, not inlined.
Runtime Sync
Deploy to ~/.roxabi/forge/ via make deploy. Shared CSS/JS mirrors for linked assets.
