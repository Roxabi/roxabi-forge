Getting Started with Forge

HTML visual artifacts for Claude Code. Diagrams, galleries, guides, epics, and presentation decks. Single-file outputs that work anywhere.

What is Forge?
Forge is a Claude Code plugin that generates self-contained HTML visual artifacts. No build step, no runtime dependencies, no server required. Output files work with file:// URLs and can be shared as attachments or deployed to any static host.
Features
Offline-First
Single-file HTML with all CSS/JS inlined. Works without network, perfect for file:// URLs.
Brand Themes
Multiple aesthetic presets. Roxabi Gold, Lyra Orange, Terminal Green, Blueprint, Editorial, Caveman.
Native Diagrams
fgraph templates for hub-and-spoke, linear flows, layered architecture, timelines, and more.
Image Galleries
Filterable galleries with starring, tags, search, and lightbox. Responsive grid with size controls.
Multi-Tab Guides
Split-file architecture with lazy-loaded tabs. User guides, architecture docs, project recaps.
Presentation Decks
Scroll-snap slide decks. Keyboard navigation, touch swipe, reveal animations. Magazine-quality output.

6 Generation Skills

Each skill targets a specific output type. Trigger by name or natural language.

Skill Overview
Forge provides 6 specialized skills for different visual output types. Each skill follows the Frame → Structure → Style → Deliver pipeline, ensuring consistent quality and brand compliance.
Available Skills
Skill Summary Triggers
forge-chart single-file fgraph visual "draw" | "diagram" | "visualize" | "quick visual"
forge-guide split-file multi-tab doc "write a guide" | "create a guide" | "multi-tab doc"
forge-epic issue-linked visual analysis "visualize #N" | "preview #N" | "epic preview"
forge-slides scroll-snap presentation deck "create deck" | "slides from #N" | "pitch deck"
forge-md render .md as-is to HTML "render md" | "md to html" | "preview md"
forge-gallery image/audio gallery "create gallery" | "image gallery" | "audio gallery"

Output Paths

How Forge outputs are generated, stored, and deployed.

Distribution Flow
Forge outputs are generated to ~/.roxabi/forge/<project>/. A local dev server watches for changes and pushes SSE live-reload events. Deploy to Cloudflare Pages via make deploy.
Generation Flow
